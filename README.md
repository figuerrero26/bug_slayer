# bug_slayer(TEAM)
BIENVENDIDOS AL PROYECTO WEB CONIITI 2026:
PRESENTACIÓN:
Este proyecto comprende el diseño y despliegue de la plataforma web para la Semana de Ingeniería 2026. La solución emplea un stack tecnológico completo (Full Stack), optimizando la gestión de datos y la interfaz de usuario para destacar la participación de Italia como nación invitada. Se enfoca en la escalabilidad, la seguridad y una experiencia de usuario temática y envolvente.ia.
